## res

The `res` folder contains the resources used by PyMODA.

| Subfolder | Contents |
| ----- | ----- |
| `colours` | Colourmaps | 
| `img` | Images | 
| `layout` | Layout files which are produced by Qt Designer | 
| `data` | Data files which are used to test the program |
